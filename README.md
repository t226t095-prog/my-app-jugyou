# Monster Defense

四方から迫り来るモンスターを撃退するシンプルかつ爽快なアクションゲームです。

## 遊び方
- **移動**: WASD または 矢印キー
- **照準**: マウス移動
- **攻撃**: マウスクリック
- **目標**: モンスターに捕まらないように倒し続け、ハイスコアを目指します。

## GitHub Pages での公開方法
1. このリポジトリを GitHub にプッシュします。
2. GitHub のリポジトリ設定（Settings）を開きます。
3. 左側のメニューから 「Pages」 を選択します。
4. 「Build and deployment」 の 「Source」 で 「Deploy from a branch」 を選択します。
5. 「Branch」 で `main`（または `master`）ブランチを選択し、フォルダを `/ (root)` に設定して 「Save」 をクリックします。
6. 数分後、提供された URL でゲームが公開されます。
